Inicializando o repositorio local

-> git init

Verificando alterações nos arquivos

-> git status

Adcionando arquivos ao controle de versão

-> git add nomeArquivo
-> git add .

Criando versões do codigo com commit

-> git commit  -m "Criando arquivo index.html com lista de curso"

Enviar para o repositorio na nuvem

-> git push

Modificar cofigurações do git

-> git config -- local user.name / user.email
-> git config -- global

Verificar o historicos de modificações no repositorio

-> git reflog

Como navegar entre versões do codigo

-> git reset --hard "ID do reflog"

Criando branches 

-> git branch "nomeBranch"

Navegando entre as branchs

-> git checkout "nomeBranch"
